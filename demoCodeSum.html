Code Summarization

For Code Summarization, we do an experiment on the dataset of 7584 pairs of comment and code of the TLCodeSum dataset in the clean version of CAT benchmark (See paper). We evaluate the effectiveness of PropMiner to the original embedding of GraphCodeBERT in this evaluation. We use Top-K accuracy as the evaluation metric for each code search phrase. In each query and candidate, we display them by single line (we mark '\n' character as ' _EL_ ' character).

Result (RQ2): GraphCodeBERT+PropMiner won 1112/7584 cases, while the original GraphCodeBERT won 450/7584 cases, leads our Mean Reciprocal Rank to be 74.65% over 76.65% of GraphCodeBERT on TLCodeSum dataset
